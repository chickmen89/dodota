# dodota

두근두근타운(두두타) 게임 편의 도구 모음

## 유틸리티 목록

### 레벨/티켓 계산기
- 레벨(2~37) 선택 시 달성일차, 티켓 현황 자동 계산
- 기준일 설정으로 달성예정일 확인
- 7종 취미(원예/요리/낚시/곤충채집/새관찰/고양이/강아지) 레벨 배분 시뮬레이션
- 티켓 부족 시 취미 레벨 선택 자동 제한
- 설정값 저장/불러오기 (localStorage)
